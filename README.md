# **Service Intro**

## 잊지 말아야할 것들을 너무 쉽게 잊어버리는 요즘. 

프로젝트 서비스의 핵심기능은`잊지 말아야 할 일들 그리고 잊혀져가는 일들을 단순히 후원 한번에 그치지 않고, 내용과 날짜를 지속적으로 떠올리게 해주는 것이 다른 서비스와의 차별점이자 가장 대표적인 기능`입니다.
